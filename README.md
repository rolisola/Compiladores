# Compiladores

Repositório para a disciplina de Compiladores do Bacharel de Ciências da Computação, turma 2025. Os arquivos neste ambiente são partes da criação de um compilador, envolvendo, por enquanto, a análise léxica e sintática de sentenças, a levando para formas pós-fixadas.

No arquivo Makefile, terão automações para a compilação do projeto, bem como a extração e compactação para zips .tar.gz. Em caso de depuração de tokens, deixamos campos para serem visíveis no terminal, ao invés de depender de debuggers tradicionais; e erros são marcados com linha e coluna, a fim de facilitar na correção de código.
